AIO FAQ - 2016-07-22

I do not know how to use this tool! :(

Download the latest AIO version, unzip the file and start ‘choose.cmd’
You have to press on keyboard “A” to “Z” and “4” to “6” for choosing tweaks and again for unchoosing.
“9” for help, “1” and “2” for switching between installation and uninstallation mode (AIO starts in installation mode),
“8” to choose all tweaks and at end “0” to start compilation of selected tweaks.
(It’s still a DOS batch, so no mouse, except for help window (“9”), here you can use cursor and mouse.)

Is AIO compatible with my FW version?

Please look into the README (section “WHICH FW VERSIONS ARE SUPPORTED”), all compatible FW versions are listed there.
But AIO will read out your FW version at beginning of installation and show if it’s not compatible (or not yet tested) and you can abort the installation at this point, if you want.

How long does it take till the installation starts?

This can take from 30 seconds to 2 minutes.

How long does the installation take?

Up to 8 minutes if every tweak has been selected (especially custom colors and speedcam-patch).

How long does the uninstallation take?

Up to 5 minutes if every tweak has been selected.

Do I have to uninstall a tweak before installing it again?

No, you can reinstall, because during installation, the tweak will be deleted and then installed again.
You can install every tweak with AIO, you never have to uninstall before.

Do I have to use same AIO version for uninstalling the tweaks?

No. You can use latest AIO version for that.

Will all previously installed tweaks automatically uninstalled, if I not choose them for new installer?

No. Only tweaks you have choosen to uninstall will be removed.

If something goes wrong, how can I help you to help me?

During installation/uninstallation, a log file is created (AIO_log.txt) and some additional files will be copied on the USB drive.
Please make a ZIP file of them (complete root of USB drive) and send that to me or upload somewhere (e.g. google drive or so) and send me the link.
You can download this tweak to copy some files from your MZD system to USB drive, that are helpful for me to.

Why the uninstaller for all tweaks has changed, so that no confirmation is needed anymore and reboot will done automatically at the end?

Because of the people who had the problem with stuck at Mazda boot logo.
If the USB ports are still working, uninstalling was not possible because the confirmation with “OK” could not be executed.

If I want to go back to original state but I forgot the tweaks I’ve installed, what can I do?

It doesn’t matter, if you choose tweaks for uninstalling, that you haven’t installed.
If the uninstaller don’t find a backup, it will copy the original files from USB drive over your original files of your MZD system, so nothing has really changed after that.
BUT: YOU SHOULD NOT DO THIS FOR NEW AND UNKNOWN FW VERSIONS!

How do I make a reboot of the MZD system?

Press NAV+MUTE for 10 seconds.

Where can I find more background images

Look at \choose\more_background-images

Is there a way to change/delete the jukebox-radio picture on the right side where album art would go?

If you don’t want to have this, then uninstall blank-album-art-frame, make an installer again and change or delete this file on your USB drive before installing:
config\blank-album-art-frame\jci\gui\common\images\no_artwork_icon.png

I don’t get GPS data after installing speedometer

This happens with newer FW versions (56.00.51x). Try this to get GPS data for these FW versions too: * After opening the doors (or trunk) at least wait 35 seconds, then start the engines * Before starting the speedometer app, shortly start the NAVI app before (works too, if NAVI app was open if engine was stopped) * Unplug and replug the NAV SD card (should be avoided)

In entertainment window, pause-on-mute works perfectly. But if you leave from entartainment window, “mute” mutes sound but not pauses song.

This behaviour is correct and as expected. The pause command doesn’t reach the player, if you are in any other app/menu.

If I take out the SD card and plug it into my computer and run the map updates on the Mazda Toolbox software will it overwrite any of the tweaks or cause issues?

No. But before you can use Mazda Toolbox you have to delete speedcam.txt/speedcam.spdb from SD-card (when you have installed speedcam-patch).
After updating, you can copy the files again.

When do i need the swapfile patch. I’ve only one usb stick with music and videos in the car. Do i need this patch then?

You don’t need it, it’s just an option to enhance the video player, if you have problems to play videos.
But I’m not sure, if it’s really worth it.

I have a question/problem with speedometer.

Please look at speedometer_readme.txt and speedometer_changelog.txt

How can I enable Hotspot on MZD system (I don’t know, if that still works with .51x FW versions):

Try the following: 1. Open diagnostics menu (buttons Music + Favorites + Pwr / Mute and hold until menu appears) 2. Hold down “DEL” until “JCI Test Mode Active” appear at right black box (15-20 sec.). 3. Press “11” and then press “Enter.” 4. Select script WiFi AP Toggle (sor so) Then the access point of the car is running, you should see a WLAN “cmu …”. Simply log in and then access via IP 192.168.53.1 on the system.

Can I install AIO even from an SD card?

No. SD card slot in the car is ONLY for satnav maps. You need to use USB port in the car, so either USB stick or USB-SD card reader with your card in it.

What are the video specs for playing videos?

Video: MP4, AVI, FLV, WMV; Audio: Stereo MPA

I can’t play videos. What can I do?

Try the following: * unplug all other USB drives, especially one, with much MP3s on it * copy only one mp3 on your video USB drive * copy only a few videos on your stick (you don’t have to delete, rename to *._mp4 is OK too, so that they will not found anymore) * reboot your system and try to play a video right after that * close the player and start again * it that works, then change the audio input after that

Is there anything to note, if I use a MAC?

If you are using a Mac or Mac formatted thumb drive make sure there are no hidden files on the drive.
The drive should be formatted Fat 32 on a PC if possible.
Look here: https://www.reddit.com/r/mazda3/comments/3d5lvr/mazda_infotainment_update/ctls5ef

What FW versions are out there?

Look at choose\docs\FW info.txt

What is the content of the speedcam.txt file?

Look at choose\docs\speedcam_info.txt

After installing date_to_statusbar mod V2.2 the top bar is full transparent and because of my bright background I can’t read the clock at all. What can I do?

Edit your background to be dark in the top, or you have to change the following png file: sh \choose\config_all\date-to-statusbar_mod\jci\gui\common\controls\StatusBar\images\StatusBarBg.png.v2.2 Rename to StatusBarBg.png, change transparency, and rename again to StatusBarBg.png.v2.2. Then install date_to_statusbar mod again.
If you want to use the original file, then look here: sh \choose\config_org_all\date-to-statusbar_mod\jci\gui\common\controls\StatusBar\images\StatusBarBg.png Copy as StatusBarBg.png.v2.2 to sh \choose\config_all\date-to-statusbar_mod\jci\gui\common\controls\StatusBar\images\ and install again

I want to change the order of the audio sources. How can I do that?

You have to edit
/jci/gui/apps/system/js/systemApp.js
line 624 - 637
Be careful, in the last line must not be a comma, or you run in to boot loop!

I know of the problems, that doing a factory reset after installing Android Auto leads the system to stay at Mazda boot loop logo, there is now AA to choose in new AIO version, but I want to install anyway. How can I do this?

You have to take a look into the readme, changelog for V1.48:
Update of “Android Auto” Headunit App to V0.92B (2016-06-13), BUT TEMPORARILY NOT VISIBLE FOR INSTALLATION MODE, because of serious factory reset problems. * You can still choose AA with “r”, but an installation is currently not recommended! * By pressing “8” (select most of the tweaks), Android Auto is first also no longer selected. * Added popup message with warning text after choosing AA if you do so nevertheless.

Why there is a tweak for backup the /jci folder, but no script to restore these files?

It’s not necessary, because:
In almost all installation jobs there are backups stored as *.org. These files are used to restore the original state.
Only if these *.org files are not found then the original files are used, which are also available on the USB stick.
For apps that require completely new files for installation, like speedometer, Android Auto, castscreen receiver and videoplayer, replacing the original files from complete backup of /jci folder would not cause removal of these files.
And there are many other additional files as markers for color themes, no_more_disclaimer, etc. that would remain in system and not be deleted.
That only happens, if you use AIO for that.
So a copy of the entire /jci backup folder would replace 99% of files that were not changed from all the tweaks and this takes a lot of unnecessary time.
And not all tweaks would be replaced or removed by that precedure.

How can I join background photos for Background Rotator tweak:

Look at: PhotoJoiner.net - http://old.photojoiner.net/
Press “Select Photos” and just choose 10 backgrounds (look at choose\more_background-images to find some).
Let set “Style” as “Vertical” (upper right corner, is already default) and klick “Join Photos”.
After that your joined graphic is shown, right click on it and select “Save Image as” to save.
Now you only have to convert the *.jpg to *.png (you can use every viewer tool out there, I use Faststone Image Viewer for that) and save as background.png in folder
sh \choose\config_all\BackgroundRotator\jci\gui\common\images on your USB drive before installing this tweak with AIO. If you want to have more or less images and adjust the rate of change:
Rate of change can be changed by editing the value of “animation” (Line:25) in config/BackgroundRotator/common.css
So, you have to modify line 25:
sh animation: slide **600**s steps(**10**,end) infinite; For example, if you want to have 20 different backgrounds and show everyone only for 10 seconds: sh animation: slide 200s steps(20,end) infinite;
